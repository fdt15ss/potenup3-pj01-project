# 프로젝트 개요

본 프로젝트는 **디지털 샤머니즘(사주·운세·타로 등)** 관련 관심 증가 현상을 데이터로 확인하고, 실제 이용 행태(검색·앱 이용·리뷰 반응)를 통해 이를 검증하는 것을 목표로 한다.

아래와 같은 흐름으로 전체 분석이 진행되었으며, 각 단계는 3개의 노트북 파일에 대응된다.

---

## 1. 주제 발견

### 문제의식

* 경기·취업·미래 불안 담론이 증가하는 시기와
* 사주/운세/타로와 같은 위로·확신 지향 콘텐츠의 소비가 함께 증가하는가?

→ **“불안이 커질수록 디지털 샤머니즘 관련 관심도는 증가한다”**는 가설을 설정했다.

---

## 2. 가설 / 확인하고자 한 것

* **가설 1**: 특정 시기(코로나 이후, 사회·경제적 불안 구간)에
  사주·운세·타로 검색량이 동반 상승한다.
* **가설 2**: 실제 앱 이용자 리뷰에서도
  ‘불안·위로·확신·의존’과 관련된 표현이 반복적으로 등장한다.

---

## 3. 데이터 수집

### (1) 네이버 카페 고민상담 게시판 글 제목 수집

* **파일**: `project01_데이터_수집_Selenium.ipynb`
* **목적**: 사회적·개인적 불안이 어떤 주제로 표출되는지 확인하고,
  이후 디지털 샤머니즘 관심 증가와의 연결 가능성 탐색
* **방식**:

  * Selenium을 이용해 크롬 브라우저 자동 제어
  * 네이버 카페의 고민상담 게시판 접속 후 게시글 제목 수집
* **수집 데이터**:

  * 게시글 제목 텍스트
  * 작성 시점 기준 최신 글 중심으로 수집
  * 총 816건의 게시글 제목
* **의의**:

  * 이용자들이 직접 드러낸 고민·불안 키워드를 확보함으로써
    검색 데이터 이전 단계의 ‘문제 인식’을 관찰할 수 있음

---

### (2) 네이버 데이터랩 검색 트렌드

* **파일**: `project01_데이터랩.ipynb`
* **목적**: 시간 흐름에 따른 관심도 변화 확인
* **방식**:

  * 네이버 데이터랩에서 추출한 엑셀 데이터를 pandas로 로드
  * 검색어: 사주 / 운세 / 타로
* **수집 범위**:

  * 2021년 2월 ~ 2026년 1월 (약 5년치 월별 데이터)
* **특징**:

  * 절대 검색량이 아닌 **상대 검색량 지표** 사용

---

### (3) 플레이스토어 앱 메타데이터

* **파일**: `project01_playstore.ipynb`
* **목적**: 분석 대상 앱의 성격 및 기본 정보 파악
* **방식**:

  * `google-play-scraper` 라이브러리 활용
  * 앱 패키지 ID 기반 정보 수집
* **수집 항목**:

  * 앱 설명, 평점, 리뷰 수, 업데이트 정보 등

---

## 4. 데이터 분석 및 시각화 결과

### (1) 검색 트렌드 시각화

* **그래프 설명**:

  * X축: 날짜
  * Y축: 검색어별 상대 검색량
* **이 그래프는**

  * 사주/운세/타로 키워드의 검색량 변화를
  * 시간 흐름에 따라 비교하기 위해 그린 선 그래프이다.
* **관찰 포인트**:

  * 특정 시기에 세 키워드가 동시에 상승하는 구간
  * 장기적으로 하락하지 않고 유지·반등하는 패턴

→ 이를 통해 **관심이 일시적 유행이 아닌 지속적 수요**임을 확인할 수 있다.

---

### (2) 리뷰 텍스트 해석 (정성 분석 기반)

* 리뷰에서 반복적으로 등장하는 키워드:

  * 불안, 위로, 맞다, 의지, 마음이 편해졌다 등
* 단순 오락 목적을 넘어
  **정서적 안정 수단으로 사용되는 경향**이 드러남

---

## 5. 결론

* 검색 데이터와 실제 이용자 리뷰를 종합했을 때,
  **디지털 샤머니즘은 사회적 불안과 밀접하게 연결된 콘텐츠 소비 형태**로 볼 수 있다.
* 가설에서 설정한

  > “불안 증가 → 운세·사주 관심 증가”
  > 는 데이터 시각화와 사용자 반응을 통해 일정 부분 확인되었다.
